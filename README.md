# Tutor Python Interactivo

Aplicativo web simple para estudiar Python desde el celular.

## Cómo publicarlo en GitHub Pages

1. Sube `index.html` a tu repositorio.
2. En GitHub entra a:
   Settings → Pages
3. En "Build and deployment":
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
4. Guarda.
5. GitHub entregará un link del tipo:
   https://nmillana.github.io/Python/

## Uso

- Abrir desde celular.
- Presionar "Empezar práctica".
- Escribir código desde cero.
- Presionar "Revisar mi código".
- Corregir y avanzar por niveles.

## Características

- Pantalla inicial.
- Lecciones agrupadas por nivel.
- Modo escribir desde cero.
- Pistas progresivas.
- Solución bloqueada hasta 2 intentos.
- Barra de avance.
- Logros.
- Repaso de errores frecuentes.
- Editor optimizado para móvil.
